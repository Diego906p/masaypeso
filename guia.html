🎯¿Qué es?

Una app educativa para practicar suma y resta de masas (kilogramos y gramos) mediante problemas con sentido real: compras en el mercado, recetas de cocina, la mochila del colegio o la bolsa de frutas. Cada problema se acompaña de un modelo de barras y una pizarra para razonar el procedimiento.

💡 Los objetos de cada problema siempre se relacionan entre sí, para que el razonamiento tenga lógica.

🦄Cómo jugar

  1. En la pantalla de inicio, toca el perfil de Luanna.
  2. Elige una fase habilitada (las bloqueadas tienen 🔒).
  3. Lee el problema. Usa la pizarra para hacer tus cuentas y el modelo de barras como guía visual.
  4. Toca 💡 Pista si necesitas ayuda, luego elige una de las 4 opciones (A–D).
  5. Pulsa Siguiente ›. Al terminar los 10 niveles verás tu calificación (AD, A, B o C).

Para salir de un ejercicio y volver a elegir fase, usa la flecha ‹ arriba a la derecha.

🎨La pizarra (pruébala)

Dibuja, traza formas, cambia el papel y haz zoom. Esta es una versión de práctica idéntica a la del juego:

En el juego además puedes borrar 🧽, deshacer/rehacer ↩️↪️, acercar/alejar ➕➖ y desplazar ✋ para tener más espacio.

📊Modelo de barras

Las barras muestran cada peso para componer (sumar) y descomponer (restar). Al iniciar el problema se llenan de forma fluida; la barra con ? se completa cuando respondes.

🔐Panel de Papá

Desde el inicio, toca Papá e ingresa la contraseña. Allí puedes:

  • En vivo: ver qué está resolviendo en este momento.
  • Historial: revisar cada respuesta y su pizarra, agrupado por día.
  • Estadísticas: precisión, aciertos y tiempo, filtrables por día.
  • Fases: activar o desactivar fases con un botón de encendido.
